## my-storybook

In the project directory, you can run:

### 실행
```bash
yarn storybook
```
Runs the app in the development mode.<br />
Open [http://localhost:9009](http://localhost:9000) to view it in the browser.

### 빌드
```bash
yarn build-storybook
```
